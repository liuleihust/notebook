## 二叉树的边界

> 递归或者 先求左边界 后求右边界 ，在用前序遍历找叶子节点，叶子节点不包含在左右边界中。

```java
class Solution {
    public List<Integer> boundaryOfBinaryTree(TreeNode root) {
    List<Integer> res = new ArrayList<>();
    if (root == null) return res;
    res.add(root.val);
     // 左边界
    helper(root.left, true, false, res);
     // 右边界
    helper(root.right, false, true, res);
    return res;
  }
  void helper(TreeNode node, boolean leftbd, boolean rightbd, List<Integer> res) {
    if (node == null) return;
      // 叶子节点
    if (node.left == null && node.right == null) {
      res.add(node.val);
      return;
    }
    if (leftbd) res.add(node.val);
      // 如果左节点不为空
    helper(node.left, leftbd && node.left !=null, rightbd && node.right == null, res);
      // 如果左节点为空，则向右
    helper(node.right, leftbd && node.left == null, rightbd && node.right != null, res);
    if (rightbd) res.add(node.val);
  }
}
```

### 二叉树的垂直遍历

>1. 先求二叉树的宽度
>2. 然后计算每个节点的索引位置
>3. 按照BFS 的方式查找

```java
class Solution {
    public  int leftmax=0;
    public  int rightmax=0;
    public List<List<Integer>> verticalOrder(TreeNode root) {
        List<List<Integer>> res = new ArrayList<>();
        if(root==null)
            return res;
        // 计算 左右长度
        countChildren(root,0,0);
        // 生成结果序列
        for(int i=0;i<leftmax+rightmax+1;i++){
            res.add(new ArrayList<>());
        }
        // 记录每个节点的索引 [0，leftmax+rightmax]
        Map<TreeNode,Integer> node2Index = new HashMap<>();
        // root 节点的索引是 leftmax
        node2Index.put(root, leftmax);
        // 层次遍历
        Queue<TreeNode> q = new LinkedList<>();
        q.offer(root);
        while(!q.isEmpty()){
            int size = q.size();
            for(int i=0;i<size; i++){
                TreeNode tn = q.poll();
                // 取出节点索引
                int index = node2Index.get(tn);
                // 添加索引
                res.get(index).add(tn.val);
                if(tn.left!=null){
                    q.add(tn.left);
                    node2Index.put(tn.left,index-1);
                }
                if(tn.right!=null){
                    q.add(tn.right);
                    node2Index.put(tn.right,index+1);
                }
            }
        }
        return res;
    }

    private void countChildren(TreeNode root,int leftindex,int rightindex){
        if(root==null)
            return ;
        countChildren(root.left,leftindex+1,rightindex-1);
        leftmax = Math.max(leftmax,leftindex);
        rightmax = Math.max(rightmax,rightindex);
        countChildren(root.right,leftindex-1,rightindex+1);
    }
}
```

### 栈排序

用一个栈实现对另一个栈的排序。将该栈从顶到底按照从大到小的顺序排序。

> 将要排序的栈记为Stack，申请的辅助栈记为help,在stack 上执行pop 操作，弹出的元素为cur。
>
> 1. 如果cur 小于或等于help的栈顶元素，直接压入 help
> 2. 否则，将help 逐一弹出，压入stack，知道cur小于等于help的栈顶元素，再将cur压入help.
> 3. 最后，将help 栈全部压入stack;

```java
Stack<Integer> help = new Stack<>();
while(!stack.isEmpty()){
    int cur = stack.pop();
    while(!help.isEmpty()&&help.peek()>cur){
        stack,push(help.pop());
    }
    help.push(cur);
}
while(!help.isEmpty()){
    stack.push(help.pop());
}
```

### 最长回文子序列

> 动态规划

```java
class Solution {
    public int longestPalindromeSubseq(String s) {
        if(s==null||s.length()==0)
            return 0;
        int n = s.length();
        int[][] dp = new int[n][n];
        for(int i=n-1;i>=0;i--){
            dp[i][i]=1;
            for(int j=i+1;j<n;j++){
                if(s.charAt(i)==s.charAt(j)){
                    dp[i][j]=dp[i+1][j-1]+2;
                }else{
                    dp[i][j] = Math.max(dp[i+1][j],dp[i][j-1]);
                }
            }
        }
        return dp[0][n-1];
    }
}
```

### 最长有效括号

> 动态规划、
>
> 我们定义一个 dp 数组，其中第i 个元素表示以下标为i的字符结尾的最长有效子字符串的长度。很明显有效的子字符串一定以")"结尾，以"("结尾的子字符串对应的dp数组位置上的值必定为0。
>
> 1. s[i]=')'且s[i-1]=‘(’，也就是字符串形如“...()”，我们可以推出：
>
>    ​				dp[i]=dp[i-2]+2
>
> 2. 如果s[i]=')'且s[i-1]=‘)’，也就是字符串形如"...))"，如果s[i-dp[i-1]-1]='(',那么
>
>    ​		dp[i]=dp[i-1]+dp[i-dp[i-1]-2]+2;
>
> 

```java
public class Solution {
    public int longestValidParentheses(String s) {
        int maxans = 0;
        int dp[] = new int[s.length()];
        for (int i = 1; i < s.length(); i++) {
            if (s.charAt(i) == ')') {
                if (s.charAt(i - 1) == '(') {
                    dp[i] = (i >= 2 ? dp[i - 2] : 0) + 2;
                } else if (i - dp[i - 1] > 0 && s.charAt(i - dp[i - 1] - 1) == '(') {
                    dp[i] = dp[i - 1] + ((i - dp[i - 1]) >= 2 ? dp[i - dp[i - 1] - 2] : 0) + 2;
                }
                maxans = Math.max(maxans, dp[i]);
            }
        }
        return maxans;
    }
}

```

> 用栈，栈中存放索引的值
>
> 对于遇到的每个‘(’ ，我们将它的下标放入栈中。
> 对于遇到的每个‘)’ ，我们弹出栈顶的元素并将当前元素的下标与弹出元素下标作差，得出当前有效括号字符串的长度。通过这种方法，我们继续计算有效子字符串的长度，并最终返回最长有效子字符串的长度

```java
public class Solution {

    public int longestValidParentheses(String s) {
        int maxans = 0;
        Stack<Integer> stack = new Stack<>();
        stack.push(-1);
        for (int i = 0; i < s.length(); i++) {
            // 只有左括号才进栈
            if (s.charAt(i) == '(') {
                stack.push(i);
            } else {
                stack.pop();
                if (stack.empty()) {
                    stack.push(i);
                } else {
                    maxans = Math.max(maxans, i - stack.peek());
                }
            }
        }
        return maxans;
    }
}
```

### 最长连续递增序列

给定一个未经排序的整数数组，找到最长且**连续**的的递增序列。

```java
class Solution {
    public int findLengthOfLCIS(int[] nums) {
        int res = 0, cnt = 0;
        for (int i = 0; i < nums.length; i++) {
            //i==0主要是结果返回res,所以1==0也走这个循环
            if (i==0 || nums[i - 1] < nums[i]) {
                //中断之前的结果存起来,和中断之后重新计数的结果取最大值
                res = Math.max(res, ++cnt);
            } else {
                //如果中断连续,重新开始计数
                cnt = 1;
            }
        }
        return res;
    }
}
```

### 最长上升子序列

>给定一个无序的整数数组，找到其中最长上升子序列的长度

>```
>input: [0, 8, 4, 12, 2]
>dp: [0]
>dp: [0, 8]
>dp: [0, 4]
>dp: [0, 4, 12]
>```

```java
public class Solution {
    public int lengthOfLIS(int[] nums) {
        int[] dp = new int[nums.length];
        int len = 0;
        for (int num : nums) {
            int i = Arrays.binarySearch(dp, 0, len, num);
            if (i < 0) {
                i = -(i + 1);
            }
            dp[i] = num;
            if (i == len) {
                len++;
            }
        }
        return len;
    }
}
```
